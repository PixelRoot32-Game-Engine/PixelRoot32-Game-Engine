# Physics Demo Example

Demonstrates the physics system with KinematicActor and RigidActor.

## Features

- KinematicActor movement
- RigidActor with gravity
- AABB collision detection
- Sensors

## Build

```bash
# Native (PC with SDL2)
pio run -e native

# ESP32
pio run -e esp32dev
```
