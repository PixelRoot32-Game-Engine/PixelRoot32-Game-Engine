# Animated Tilemap Example

Demonstrates animated tile maps with color palette cycling.

## Features

- Animated tilemap rendering
- Color palette animation
- Scene arena usage
- 2bpp and 4bpp sprite rendering

## Build

```bash
# Native (PC with SDL2)
pio run -e native

# ESP32
pio run -e esp32dev
```
