# Sprites Demo Example

Demonstrates 2BPP and 4BPP sprite rendering with animations.

## Features

- 2BPP sprites (4 colors)
- 4BPP sprites (16 colors)  
- Sprite animation

## Build

```bash
# Native (PC with SDL2)
pio run -e native

# ESP32
pio run -e esp32dev
```
